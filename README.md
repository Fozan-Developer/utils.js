## 📦 Install

**NPM**
```sh
npm instal @mr_fozan/utils.js
```
**YARN**

```sh
$ yarn add @mr_fozan/utils.js
```

## 🚀 Usage

```js
const utils = require('@mr_fozan/utils.js');
```
